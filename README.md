# Program Kasir POS4U

Project Program Kasir berbasis **CodeIgniter 4** yang dibuat untuk memenuhi penilaian **UAS** selama perkuliahan.

## Deskripsi
Aplikasi ini digunakan untuk membantu proses transaksi penjualan, pengelolaan produk, dan pencatatan data penjualan secara sederhana dan efisien.

## Fitur
- Manajemen produk
- Manajemen kategori
- Transaksi penjualan
- Laporan penjualan

## Teknologi yang Digunakan
- PHP 8.1+
- CodeIgniter 4
- MySQL
- Bootstrap
- Node.js (untuk asset build)
